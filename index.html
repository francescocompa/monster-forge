Bracket shortcuts

Type these tokens in any trait, action, reaction, or note. They expand in the preview and in every export.

[C] · [c]“The creature” / “the creature” — uses the Short name word (capitalised vs lowercase). Respects the Proper name toggle.
[Name] · [name]Aliases for [C] / [c].
[s]Verb “-s” for a singular subject; drops to nothing when Plural is on. e.g. make[s] → makes / make.
[2d6 + 6]Averages any dice expression on render → 13 (2d6 + 6).
[STR SAVE] · [SAVE]Save DC for that ability (8 + PB + modifier) → e.g. DC 12. Any ability: STR/DEX/CON/INT/WIS/CHA. Bare [SAVE] uses the creature's highest ability.
[STR ATK] · [ATK]Attack / check modifier for that ability (PB + modifier) → e.g. +4. Bare [ATK] uses the creature's highest ability.
[C+] · [c+] · [C-] · [c-]Override the article for one reference: + adds "the", - removes it — ignoring the Proper name toggle. Capital letter = capitalised.
{C} · {c} · {s}Brace forms are accepted too, in case square brackets clash with your text.
